## 开发方式

所有项目均采用 **Vibe Coding** 工作流开发：

1. **需求拆解** → 将需求分解为明确的功能模块
2. **AI 生成** → 使用 Trae AI 生成代码框架和主要逻辑
3. **人工审查** → 审核代码质量、逻辑正确性、安全性
4. **测试交付** → 运行验证、编写文档、打包交付

## 快速开始

每个项目独立运行，进入对应目录即可：

```bash
cd data_cleaner
pip install -r requirements.txt
python data_cleaner.py -i ./data
```
